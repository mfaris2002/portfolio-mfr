# Strapi CMS Backend

This folder contains the Strapi CMS for your portfolio.

## Getting Started

1.  Navigate to this folder: `cd backend`
2.  Start the development server: `npm run develop`
3.  Open the admin panel: [http://localhost:1337/admin](http://localhost:1337/admin)

## Integration with Nuxt

The Nuxt project is already configured to connect to this backend via `@nuxtjs/strapi`.
URL: `http://localhost:1337` (configured in `.env`)
